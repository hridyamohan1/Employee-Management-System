# 🧑‍💼 Employee Management System

## 📌 Overview

The **Employee Management System** is a software application designed to manage employee data efficiently. It helps organizations handle employee records, roles, attendance, and other essential details in a centralized system.

---

## 🚀 Features

* Add new employees
* Update employee details
* View employee list
* Search employees
* Manage roles and departments
* Add Salary

---

## 🛠️ Technologies Used

* **Frontend:** React.js
* **Backend:**  Python Django Rest Framework
* **Database:**  Sqlite3,Django ORM
